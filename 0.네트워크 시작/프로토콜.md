# 프로토콜
네트워크의 통신 규약.  

1900년대에 네트워크가 갓 등장했을 때는 가용자원도 속도도 떨어지고,   
자연어도 처리할 수 없는 열악한 환경이었다.  

프로토콜은 대부분 문자가 아닌 2진수 비트 기반으로 만들어졌다. 

&nbsp;

프로토콜 기술도 복잡하고 흩어져 있엇지만,  
이더넷-TCP/IP 기반으로 표준화되고 있다.  

- 물리적 측면(데이터 전송 매체, 신호 규약, 회선 규격 등) :  
이더넷

- 논리적 측면(장치들끼리 통신하기 위한 프로토콜 규격) :  
TCP/IP
